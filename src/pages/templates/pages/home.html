Ninja News Time Machine

This service collects news from YouTube videos and provides their transcripts and an option to summarize the transcript using the GPT-3.5-Turbo LLM model. This service also provides a chat interface powered by the same model where users can ask questions about the news.

Features:

  • Collects news from YouTube videos
  • Provides transcripts of the news
  • Summarizes the transcript using the GPT-3.5-Turbo LLM model
  • Provides a chat interface powered by GPT-3.5-Turbo
